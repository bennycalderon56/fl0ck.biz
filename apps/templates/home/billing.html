Useful Technical Indicators

  • Simple Moving Averages

    The Simple Moving Average (SMA) is the average of a selected range of prices(usually closing price) by the number of periods in that range, in this case 200 days.

    A good entry using SMA would be the 2nd highlighted red circle, before it breaks above it.

    SMA = (A1 + A2 + A3 +...+ An)/n
    An = price of stock at period n
    n = total number of periods

  • Exponential Moving Averages

    The Exponential Moving Average (EMA) is the moving average with greater weight it uses more relevant data than SMA.

    A good entry using EMA would be the both highlighted circles. In this case the 4 EMA crossed up the 50 EMA resulting in a pop up.

    EMA = Closing Price * (n) + EMA(Previous Day) * (1-n)
    n = total number of periods

  • Traingular Moving Averages

    The Triangular Moving Average (TMA) is an average of an average, of the selected prices
    It acts as a very clear bullish or bearish indicator, if the current price is above then it has bullish movement, while if the current price is below then it is bearish until it crosses above its TMA.

    Since its an average of an average we:
    SMA = (P1 + P2 + P3 + ... + Pn) / n
    TMA = (SMA1+ SMA2+ SMA3+ ... + SMAn) / n
    P = Closing Price of Stock
    n = total number of periods

  • Volume

    Volume is the number of shares of a security traded during the given period of time.
    Volume is a major key since it is used to measure the relative significance of a market move.
    The more volume there is the higher the stock can move.
    The lower volume there is the less a stock can move.

  • Volume Weighted Average Price

    The Volume Weighted Average Price (VWAP) helps in giving the average price a stock has traded throughout the day using both volume and price.

    1 way of using the VWAP is to try and buy below the VWAP then sell above the VWAP
